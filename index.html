Install Cosmic

Download once. Run Cosmic like a normal Windows app.

The installer bundles the desktop app, required internal files, plugin repository, updater files, and security modules. Users only download Cosmic_Beta_1.0.0_RC1_Setup.exe. Online AI and Screen Vision use Cosmic Server when configured; local commands and tools still work offline on the PC.

1

Download

Click the download button and save Cosmic_Beta_1.0.0_RC1_Setup.exe.

2

Install

Run the installer, allow Windows permissions if prompted, and finish setup.

3

Start

Open Cosmic from the Desktop shortcut or Windows Start menu.

4

Test

Type in chat to check the release package.
