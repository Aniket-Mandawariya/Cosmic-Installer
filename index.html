Install Cosmic

Download once. Run Cosmic like a normal Windows app.

The installer bundles the desktop app and required internal files. Users only download CosmicSetup.exe. Online AI and Screen Vision use Cosmic Server when configured; local commands and tools still work on the PC.

1

Download

Click the download button and save CosmicSetup.exe.

2

Install

Run the installer, allow Windows permissions if prompted, and finish setup.

3

Start

Open Cosmic from the Desktop shortcut or Windows Start menu.

4

Test

Type in chat to check everything.
