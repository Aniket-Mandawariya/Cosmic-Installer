# Cosmicweb

Static download website for Cosmic AI Assistant.

Open `index.html` in a browser to preview it. The download button points to the latest GitHub Release:

```text
https://github.com/Aniket-Mandawariya/Cosmic-Installer/releases/latest/download/CosmicSetup.exe
```

To publish the site, upload this folder to your hosting provider. Upload `CosmicSetup.exe` to GitHub Releases because normal GitHub repo files cannot be larger than 100 MB.
