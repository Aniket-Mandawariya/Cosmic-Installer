# Downloads

The Windows installer is not stored in this repo because `CosmicSetup.exe` is larger than GitHub's normal 100 MB file limit.

```text
CosmicSetup.exe
```

The website download buttons already point to:

```text
https://github.com/Aniket-Mandawariya/Cosmic-Installer/releases/latest/download/CosmicSetup.exe
```
